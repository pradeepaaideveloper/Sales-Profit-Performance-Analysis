# Sales and Profit Performance Analysis Dashboard
## Project Overview

This project presents an interactive Sales and Profit Performance Analysis Dashboard developed using Power BI. The dashboard provides valuable insights into business performance by analyzing sales, profit, customer behavior, product categories, and regional trends.

The dataset was cleaned and transformed using Excel and Power Query before being loaded into Power BI for data modeling and visualization. DAX measures and calculated columns were created to derive key performance indicators and support interactive analysis.

## Key Features

* KPI Cards for Total Orders, Total Sales, Total Profit, and Total Quantity
* Sales Level and Profit Level slicers for dynamic filtering
* Regional order analysis
* Category-wise sales and profit comparison
* Customer segment sales distribution
* Customer type profit analysis
* Monthly sales and profit trend analysis
* Interactive dashboard design with user-friendly navigation

## Tools & Technologies

* Microsoft Excel
* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* GitHub

## DAX Measures Used

* Total Orders = COUNTROWS()
* Total Sales = SUM()
* Total Profit = SUM()
* Total Quantity = SUM()

## Calculated Columns

* Sales Level (High Sale / Good Sale)
* Profit Level (High Profit / Good Profit)

## Business Insights

The dashboard helps identify high-performing regions, profitable product categories, customer purchasing patterns, and monthly performance trends, enabling data-driven decision-making.

## Project Outcome

This dashboard demonstrates end-to-end Business Intelligence skills, including data cleaning, data modeling, DAX calculations, dashboard design, and business insight generation using Power BI.
